# 12v-Power-Supply
This is a 12 volts power supply circuit.
We used the bridge diode for the rectification section.
<br>
### COMPONENTS USED
- Bridge rectifier diode
- 1000uf Capacitor x2
- Resistor 1k
- LED
- Voltage Regulator LM7812
- DC Jack
- Switch
- Connector
